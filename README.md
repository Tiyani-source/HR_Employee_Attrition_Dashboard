# HR_Employee_Attrition_Dashboard


## Overview

This project presents a complete Power BI dashboard built to analyze employee attrition using a large HR dataset (1500+ employee records).
The dashboard provides insight into attrition patterns, employee demographics, job satisfaction levels, department performance, and education field trends.

The repository includes both the Excel dataset and the final Power BI (.pbix) file.

## Tools 
- Power BI 
-	Microsoft Excel

## Dataset 

The dataset contains information for more than 1500 employees, including fields such as:
-	Age
-	Gender
-	Department
-	Job Role
-	Business Travel
-	Education & Education Field
-	Work Experience metrics
-	Monthly Income
-	Environment Satisfaction
-	Work-Life Balance
-	Job Satisfaction
-	Years at Company
-	Years Since Last Promotion
-	Attrition (Yes/No)

The dataset is provided in the repository as HR_Employee_Data.xlsx.



## Dashboard Features
1. Employee Overview (KPIs)
	-	Overall Employees
	-	Attrition Count
	-	Attrition Rate (%)
	-	Active Employees
	-	Average Age
2. Department-Wise Attrition
- A pie chart showing attrition distribution across HR, Sales, and R&D departments.
3. Employees by Age Group and Gender
- A clustered column chart displaying employee count across different age ranges, split by gender.
4. Job Satisfaction Rating Matrix
- Matrix visual summarizing satisfaction levels (1 to 4) across different job roles.
5. Education Field Attrition
- Horizontal bar chart showing attrition count by education domain.
6. Attrition by Age & Gender

Donut charts showing gender-wise attrition distribution across age bands:
-	Under 25
-	25–34
-	35–44
-	45–54
-	Over 55
